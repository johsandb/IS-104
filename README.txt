Oppg 1

Canvas er spesielt nyttig til applikasjoner som trenger høy ytelse, og lite grafikk. Html/css er bedre på text
de gjør det enkelt, og kan style det opp enkelt også. Canvas bruker lengre tid på å rendre og kan bare ha en style regel
som gjør det ganske begrenset. 

Html/css har en meningsfull struktur, mens canvas har ikke. Svaksynte eller blinde kan heller ikke bruke canvas pga 
strukturen. På smarttelefoner vil canvas se akkurat likt ut på alle telefoner som kan være en positiv og negativ ting,
mens html vil fort tilpasse seg brukerens mobil og stole på at oppsettet blir riktig da. De fleste typer rendring vil
fungere bedre i canvas. Rett og slett rendre fortere. Canvas er også nyttig visst du bruker det på flere plattformer
feks smarttelefon og nettbrett vil det med canvas se helt likt ut, mens med HTML kan det bli problemer.
